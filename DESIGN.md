# UI-kit RoundDate

## 1. Визуальная идея

Стиль проекта - мягкий премиальный consumer-интерфейс для офлайн-знакомств. Основа: теплый светлый фон, коралловые акценты, крупная типографика, pill-кнопки, glass-карточки, мягкие тени и 3D/фото-ассеты.

Референсы в `references/` имеют единый формат `1536x1024` и один дизайн-язык: молочно-розовый фон, насыщенный coral/red CTA, черные заголовки, серо-синие вторичные тексты, полупрозрачные карточки с тонким розовым бордером.

## 2. Цвета

### Brand

- `--sd-coral-500: #ff5a52` - основной бренд-акцент, иконки, ссылки, бейджи.
- `--sd-coral-600: #fc4238` - активная часть градиента кнопок.
- `--sd-coral-400: #ff756d` - hover/soft акцент.
- `--sd-coral-100: #ffe8e4` - мягкие фоны бейджей и иконок.
- `--sd-coral-050: #fff4f2` - общий blush background.

### Text

- `--sd-ink-900: #080c0f` - основные заголовки.
- `--sd-ink-800: #111317` - основной текст.
- `--sd-ink-600: #5f6973` - вторичный текст.
- `--sd-ink-500: #7b828a` - muted подписи.
- `--sd-white: #ffffff` - текст на ярких кнопках.

### Surfaces

- `--sd-bg: #fff6f4` - фон страницы.
- `--sd-surface: rgba(255, 248, 246, 0.82)` - карточки.
- `--sd-surface-strong: rgba(255, 255, 255, 0.9)` - поля, кнопки outline.
- `--sd-border: rgba(255, 113, 97, 0.18)` - тонкие границы.
- `--sd-border-strong: rgba(255, 113, 97, 0.28)` - активные границы.

## 3. Типографика

Основной шрифт: `Manrope`.

Причина: хорошо работает с кириллицей, выглядит мягко и современно, близок к референсу по геометрии и плотности.

Размеры:

- Display XL: `76px / 1.08 / 800` - главный hero-заголовок на desktop.
- Display LG: `64px / 1.08 / 800` - заголовки секций.
- Heading MD: `34px / 1.15 / 800` - карточки и крупные числа.
- Heading SM: `24px / 1.25 / 800` - названия карточек.
- Body LG: `22px / 1.45 / 500` - hero-subtitle.
- Body MD: `18px / 1.55 / 500` - основной текст.
- Body SM: `15px / 1.45 / 600` - бейджи, подписи, навигация.

## 4. Радиусы

- Small: `14px` - бейджи, маленькие chips.
- Medium: `22px` - поля и компактные кнопки.
- Large: `28px` - большие карточки.
- Pill: `999px` - CTA-кнопки, nav button, chips.

## 5. Тени

- CTA: `0 16px 28px rgba(252, 66, 56, 0.24)`.
- Card: `0 24px 70px rgba(170, 91, 79, 0.08)`.
- Soft object: `0 30px 80px rgba(170, 91, 79, 0.14)`.

## 6. Button

Кнопки всегда pill-shaped. Основной CTA - coral gradient, белый текст, стрелка справа. Вторичная кнопка - белый/молочный фон, coral border, темный текст. Disabled-состояние снижает opacity и убирает pointer events.

Размеры:

- `sm`: height 40, padding 16.
- `md`: height 48, padding 22.
- `lg`: height 56, padding 28.
- `hero`: height 62, padding 34, min-width 298.
- `icon`: square 44.

## 7. Карточки

Карточки используют glass-effect:

- фон `rgba(255, 248, 246, 0.82)`;
- border `1px solid rgba(255, 113, 97, 0.16)`;
- blur `18px`;
- radius `28px`;
- shadow `0 24px 70px rgba(170, 91, 79, 0.08)`.

## 8. Анимационный принцип

Анимации должны быть мягкими, без резких bounce-эффектов. Подходящие движения:

- медленный float для 3D-объектов;
- легкий parallax относительно скролла;
- micro-tilt при движении курсора;
- плавный shimmer/свечение у CTA;
- staggered reveal карточек.
