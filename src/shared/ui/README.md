# Shared UI

Custom design-system components will live here. Use Radix only for low-level accessible
behavior where needed.

## Components

- `Button` - pill-shaped button component based on the reference UI-kit.
- `Modal` - Radix-based modal window with overlay, close button, controlled and uncontrolled modes, and size variants up to `xl`.
- `Input` - text input with label, label action, hint/error states, icons and reusable sizes.
- `Select` - custom listbox-style selector for reusable option fields.
- `DatePicker` - custom pink calendar picker with month and year controls.
- `Switch` - controlled semantic switch for binary settings.
- `ScrollArea` - reusable thin vertical scroll container for dense panels.
